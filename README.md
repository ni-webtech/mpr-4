Embedthis Multithreaded Portable Runtime (MPR)
==============================================

To build, you will need to install Ejscript and Built It (bit). See:
    https://github.com/embedthis/ejs-2
    https://github.com/embedthis/bit

To build:
<pre>
    configure
    bit
</pre>
